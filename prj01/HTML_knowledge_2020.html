Today I'll show some things that I know about HTML programming.

First

For the first demonstration, I'll create a "thank you" note from a self-service cashier from a random store.
So, what you'll need to do is simply insert your name on the input space and press the "done" button.





Next

For this next demonstration, I'll create some basic mathematical functions. First, click on the button
of the function you desire. Then fill the two/three boxes with valid values (in a variables box, insert
a letter). Lastly, click submit and you will get your answer.























Last

For the last demonstration, I'll create three mathematical functions: rectangleArea, triangleArea, and circleArea.
First, you'll need to choose a function by clicking its button (by doing that, other buttons will appear).
Next, you will choose the value of the shape's dimension(s) (length and width for rectangle, base and height triangle,
radius for circle). Finally, click the "calculate" button, and your answer will appear.
